Celeste (Cocos2d-x Remake)

A Cocos2d-x 4.0 remake project (Windows)

Open-source on GitHub

A C++ game project recreating Celeste gameplay using Cocos2d-x. Includes start menu, settings, level system, character movement/jump, traps and skill effects, pause/save, and music/sound.

GitHub    Demo

Runtime Environment

  • Visual Studio: VS2019 / VS2022
  • Cocos2d-x: 4.0
  • Windows: 10 / 11

Functions

  • Game start interface, background music, and exit function
  • Keyboard for walking/jumping; mouse for settings and entering levels
  • Effects of skills and traps
  • Pause and save
  • Movement display
